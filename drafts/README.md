# 草案与设计稿归档

本目录用于集中存放尚未定稿的草案、设计说明或未来功能的探索性文档，避免将草案内容直接写入根目录 README。

## 使用约定
- 每个草案单独建文件，命名示例：`2024-07-privileged-backup-rules.md`。
- 在开头注明作者、日期与背景，标识当前状态（草案/评审中/已采纳/弃用）。
- 草案定稿后，可将结论整合到正式文档（如根 README 或相关子目录 README），并在此处保留指向正式文档的链接。
- 如包含示例数据或脚本，请将文件放在 `demo_data/` 等对应目录，并在草案中给出引用说明。

## 目录导航
- 目前暂无草案文件，可按需新增。
