# demo_data 目录说明

此目录用于存放示例数据与调试素材，不参与正式检测流程。常见文件与用途如下：

- `tree.txt` / `*.txt` 路径树：提权备份目录的完整路径列表（等效 `tree -a` 输出），便于推导路径模式与优化规则。
- `sample_privileged_tree.txt`：项目内置的小型示例，展示提权备份常见层级，可用于快速对照命名与路径分布。
- 辅助脚本：可放置用于扩展能力的 Python 工具（如自定义规则解析、数据转换），方便后续整合。

使用建议
1. 文件仅供参考与调试，不会被检测流程自动读取或作为安全判断依据。
2. 提交前请做好脱敏处理（移除账号、设备标识、域名、证书等敏感信息）。
3. 若存放大体积样例，建议压缩并在此 README 中注明来源、使用方式与解压目录名，便于协作者快速复现。
4. 如需新增类型，请在此 README 中补充简要说明，保持目录结构可读性。
